# 학생 자리 배치 프로그램 (TeacherSeatSetter)

## 1. 프로젝트 개요

### 목적
선생님이 학생들의 자리를 쉽고 빠르게 배치할 수 있는 데스크톱 프로그램

### 타겟 사용자
- 초/중/고등학교 담임 교사
- IT에 익숙하지 않은 교사도 쉽게 사용 가능해야 함

### 핵심 가치
1. **자동 배치** — 랜덤 배치, 조건부 배치(남녀/시력/키 등)로 클릭 한 번에 자리 완성
2. **자유로운 교실 구성** — 드래그&드랍으로 다양한 좌석 형태를 자유롭게 배치
3. **간편한 출력** — A4 인쇄, PDF/JPG 저장으로 바로 교실에 게시 가능

---

## 2. 화면 구성

프로그램은 좌측 사이드바 네비게이션 + 우측 콘텐츠 영역으로 구성된다.
사이드바에 단계 표시(① 학생 관리 → ② 교실 설정 → ③ 자리 배치)를 제공하여 초기 사용자가 순서를 파악할 수 있도록 한다.

### 2-1. 학생 관리

학생 명단을 등록하고 관리하는 화면.

#### 기능
- **엑셀 가져오기**: 엑셀 파일(.xlsx)에서 학생 명단 일괄 가져오기
  - 샘플 엑셀 템플릿 다운로드 버튼 제공
  - 가져오기 결과 요약 표시 (성공 N명, 실패 N명, 실패 사유)
- **엑셀 내보내기**: 현재 학생 명단을 엑셀로 저장
- **직접 추가**: UI에서 학생 이름, 번호를 입력하여 직접 추가
- **편집/삭제**: 학생 목록에서 선택하여 이름, 번호 수정 또는 삭제
- **반(Class) 관리**:
  - 반 추가/이름 변경/삭제
  - 반별 학생 수 표시
- **학생 속성** (선택사항):
  - 성별 (남/여) — 남녀 분리 배치에 사용
  - 시력 메모 (앞자리 필요 여부) — 조건부 배치에 사용
  - 키 구분 (상/중/하) — 조건부 배치에 사용

#### 데이터
- 학생 데이터는 암호화하여 로컬에 저장
- 저장 항목: 반 이름, 학생 이름, 출석번호, 성별, 속성 메모

---

### 2-2. 교실 설정

교실의 좌석 배치를 설정하는 화면.

#### 좌석 종류
| 종류 | 설명 |
|------|------|
| 1인석 | 1명이 앉는 개별 책상 |
| 2인석 | 2명이 나란히 앉는 책상 |
| 1줄석 (3~8인) | 긴 테이블에 한 줄로 앉는 형태 |
| 원형석 (3~8인) | 원형/반원형 테이블에 둘러 앉는 형태 |
| 테이블석 | 직사각형 테이블 (한 면당 2명, 총 8명) |
| 극장석 | 강당형 고정 좌석 (가로 일렬 배치) |

#### 배치 방법
- **드래그&드랍 자유 배치**: 왼쪽 컴포넌트 바에서 좌석/가구를 드래그하여 캔버스에 배치
- **격자 자동 배치**: 좌석 종류 선택 후 "4 x 5" 같은 행/열을 입력하면 자동 격자 배치
- **좌석 이동/회전**: 배치된 좌석을 드래그하여 위치 변경, 우클릭으로 회전
- **좌석 삭제**: 선택 후 Delete 키 또는 삭제 버튼

#### 교실 구성 요소
| 요소 | 설명 |
|------|------|
| 칠판/화이트보드 | 교실 앞쪽 표시 (방향 기준점) |
| 강단/교탁 | 교사 위치 표시 |
| 출입문 | 교실 출입구 위치 표시 |

#### 교실 설정 저장
- 여러 개의 교실 설정을 저장/관리 가능 (예: "3학년 1반 교실", "과학실")
- 교실 이름 지정
- 교실 목록에서 선택하여 수정/삭제

### 2-3. 자리 배치

학생 관리에서 등록한 학생과 교실 설정에서 만든 좌석을 매칭하여 자리를 배치하는 화면.
이 화면이 프로그램의 **핵심 기능**이다.

#### 배치 방법

**a) 랜덤 배치**
- "랜덤 배치" 버튼을 누르면 학생을 무작위로 좌석에 배정
- 배치할 때마다 다른 결과 생성
- 학생 수 > 좌석 수일 때: 초과 학생 알림 표시
- 학생 수 < 좌석 수일 때: 빈 좌석 표시

**b) 조건부 배치** (선택사항)
- 남녀 분리 배치: 남학생/여학생을 좌우 또는 앞뒤로 분리
- 앞자리 우선 배치: 시력이 나쁜 학생을 앞쪽에 배치
- 키순 배치: 키가 작은 학생을 앞쪽에 배치
- 조건 조합 가능 (예: 남녀 분리 + 키순)

**c) 수동 배치**
- 좌측 학생 목록에서 학생 이름을 드래그하여 좌석에 드랍
- 이미 배치된 학생을 다른 좌석으로 드래그&드랍하여 자리 교체
- 좌석의 학생을 학생 목록으로 드래그하여 배치 해제
- 빈 좌석 클릭 시 미배치 학생 목록에서 선택하여 배치

**d) 혼합 배치**
- 먼저 랜덤/조건부 배치 후, 수동으로 일부 학생 위치 조정 가능

#### 배치 시각화
- 학생이 배치된 좌석: 파란색 하이라이트 + 학생 이름 표시
- 빈 좌석: 흰색/회색
- 드래그 중: 드랍 가능 영역 초록색 하이라이트, 불가 영역 빨간색
- 미배치 학생 수 표시

#### 배치 결과 저장
- 현재 배치를 저장하여 나중에 불러오기 가능
- 저장 시 날짜, 반 이름, 교실 이름 자동 포함
- 이전 배치 이력 목록 조회 가능

#### 출력
- **JPG 저장**: 배치도를 이미지로 저장
- **PDF 저장**: A4 크기 PDF로 저장
- **인쇄**: A4 용지에 직접 인쇄
- 출력물에 포함되는 정보: 반 이름, 날짜, 학기, 좌석 배치도

## 3. 데이터 관리

### 저장 구조
- 모든 데이터는 로컬 암호화 저장 (AES-128)
- 저장 위치: `%LOCALAPPDATA%\TeacherSeatSetter\`
- 저장 파일:
  - `students.dat` — 학생 명단 (반별)
  - `classrooms.dat` — 교실 설정
  - `arrangements.dat` — 자리 배치 이력

### 백업/복원
- 전체 데이터를 하나의 파일(.tss)로 내보내기/가져오기
- 학기 초에 이전 데이터 초기화 기능

---

## 4. UI/UX 원칙

1. **최소 클릭**: 핵심 작업(랜덤 배치→인쇄)을 3클릭 이내로 완료 가능
2. **시각적 피드백**: 드래그&드랍 시 하이라이트, 성공/실패 시 명확한 표시
3. **한국어 UI**: 모든 메시지, 버튼, 안내문 한국어
4. **실행 취소**: Ctrl+Z로 마지막 배치 변경 취소 가능
5. **단계 안내**: 처음 사용자를 위한 사이드바 단계 표시 (① → ② → ③)
6. **자동 저장**: 변경 사항 자동 저장, 비정상 종료 시에도 데이터 보존
7. **에러 구분**: 정보/경고/에러를 색상과 아이콘으로 구분 표시
